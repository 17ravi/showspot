# ShowSpot

A cross-platform social media app that organizes photos by location. Share your travel memories and explore a visual map of everywhere you've been.

## About

ShowSpot is a social media platform designed around location-based photo sharing. Instead of a traditional feed or profile, your presence on ShowSpot is represented as an interactive map showing all the places you've visited and the photos you've taken at those locations. Discover new destinations through other users' photo collections and build a visual travel history of your own.

## Key Features

- **Location-Based Photo Organization**: Upload photos and automatically organize them by the location where they were taken
- **Interactive Map Profile**: Your profile displays as a map of all the places you've visited with photo galleries at each location
- **Cross-Platform Support**: Seamlessly use ShowSpot on mobile, tablet, and desktop
- **Photo Discovery**: Explore photos from other users organized by location
- **Travel History**: Build a visual record of everywhere you've been and what you've captured

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/17ravi/showspot.git
cd showspot
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
